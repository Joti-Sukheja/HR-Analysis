## ðŸ“Š Key Insights  
- **Total Employees:** **1,480**  
- **Attrition Overview:**  
  - **Total Attrition:** **238 employees**  
  - **Attrition Rate:** **16.1%**  
- **Gender Breakdown:**  
  - **Male Employees Leaving:** 151

- **Attrition by Age Group:**

  - Employees with **Life Sciences degrees (39%)** have the highest attrition.
